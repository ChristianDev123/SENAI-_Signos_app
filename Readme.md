## Como instalar localmente:

Siga os seguintes passos utilizando o CMD do Windows ou Bash do Linux:

```cmd
    git clone https://github.com/ChristianDev123/SENAI_ReactJS_Signos_app.git
```
```cmd
    cd SENAI_ReactJS_Signos_app
```
```cmd
    yarn install
```
```cmd
    yarn start
```
